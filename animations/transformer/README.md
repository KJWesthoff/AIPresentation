# Transformer Animation

Create a Remotion video animation showing how transformer architecture and self-attention mechanism work.

## Suggested Animation Content

1. **Input Sequence** - Show all words in a sentence appearing at once (parallel processing)
2. **Self-Attention Mechanism**
   - Visualize how each word "attends" to all other words
   - Show attention weights as connecting lines or heat maps
   - Example: In "The cat sat on the mat", show "cat" attending strongly to "sat" and "mat"
3. **Parallel Processing** - Emphasize that all tokens are processed simultaneously (not sequentially like LSTM)
4. **Multi-Head Attention** - Show multiple attention patterns happening in parallel
5. **Encoder-Decoder** - Optional: Show information flowing from encoder to decoder
6. **Breakthrough Moment** - End with a visual flourish emphasizing the paradigm shift

## Output
- File: `transformer-animation.mp4`
- Duration: 15-20 seconds
- Dimensions: 1280x720 (or aspect ratio that fits the placeholder)

## Color Scheme
Match the presentation theme:
- Background: #1a1a2e (dark navy)
- Primary accent: #00d9ff (cyan)
- Secondary: #9d4edd (purple)
- Tertiary: #ff006e (pink) for emphasis
- Use gradients and glows to emphasize the "breakthrough" nature
