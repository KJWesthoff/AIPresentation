# CNN Animation

Create a Remotion video animation showing how convolutional neural networks process images.

## Suggested Animation Content

1. **Input Image** - Show a sample image (e.g., a cat or digit)
2. **Convolutional Layers** - Animate filters sliding across the image, showing feature maps
3. **Pooling Layers** - Show downsampling/pooling operations
4. **Feature Extraction** - Visualize how different layers detect different features (edges → shapes → objects)
5. **Final Classification** - Show the flattened layer connecting to output neurons

## Output
- File: `cnn-animation.mp4`
- Duration: 10-15 seconds
- Dimensions: 1280x720 (or aspect ratio that fits the placeholder)

## Color Scheme
Match the presentation theme:
- Background: #1a1a2e (dark navy)
- Primary accent: #00d9ff (cyan)
- Secondary: #9d4edd (purple)
