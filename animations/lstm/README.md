# LSTM Animation

Create a Remotion video animation showing how LSTM networks process sequential data.

## Suggested Animation Content

1. **Input Sequence** - Show a sequence of words or tokens flowing in one at a time
2. **LSTM Cell Structure** - Visualize the gates (forget gate, input gate, output gate)
3. **Hidden State & Cell State** - Show how information flows through the cell state
4. **Sequential Processing** - Emphasize the step-by-step nature (word 1 → word 2 → word 3)
5. **Memory Persistence** - Highlight how the cell state maintains context over time
6. **Output Generation** - Show predictions being made at each step

## Output
- File: `lstm-animation.mp4`
- Duration: 10-15 seconds
- Dimensions: 1280x720 (or aspect ratio that fits the placeholder)

## Color Scheme
Match the presentation theme:
- Background: #1a1a2e (dark navy)
- Primary accent: #00d9ff (cyan)
- Secondary: #9d4edd (purple)
- Tertiary: #ff006e (pink) for highlighting gates
