# AI Presentation

Reveal.js presentation covering the evolution of AI — from early ML and MNIST to CNNs, RNNs, and Transformers. Includes interactive D3.js neural network animations trained on real MNIST data.

## Quick start

Open `presentation.html` directly in a browser (no server needed).

---

## Assets excluded from the repository

Several large or sensitive items are not committed. Follow the steps below to restore them.

### 1. MNIST dataset (~53 MB)

The raw IDX binary files must be placed in `MNIST_ORG/`:

```
MNIST_ORG/
  train-images.idx3-ubyte
  train-labels.idx1-ubyte
  t10k-images.idx3-ubyte
  t10k-labels.idx1-ubyte
```

Download from the official mirror:

```bash
mkdir -p MNIST_ORG && cd MNIST_ORG
curl -O http://yann.lecun.com/exdb/mnist/train-images-idx3-ubyte.gz
curl -O http://yann.lecun.com/exdb/mnist/train-labels-idx1-ubyte.gz
curl -O http://yann.lecun.com/exdb/mnist/t10k-images-idx3-ubyte.gz
curl -O http://yann.lecun.com/exdb/mnist/t10k-labels-idx1-ubyte.gz
gunzip *.gz
# Rename to match expected filenames (replace hyphens with dots in extension)
mv train-images-idx3-ubyte train-images.idx3-ubyte
mv train-labels-idx1-ubyte train-labels.idx1-ubyte
mv t10k-images-idx3-ubyte  t10k-images.idx3-ubyte
mv t10k-labels-idx1-ubyte  t10k-labels.idx1-ubyte
```

> If the LeCun mirror is down, the same files are available at
> https://storage.googleapis.com/cvdf-datasets/mnist/

### 2. Training data for animations (`mnist-sample.js`, ~1.7 MB)

This file is generated by training a small MLP (784 → 64 → 32 → 10) on MNIST and serialising 50 sample images plus per-epoch weight snapshots. It is required for the D3 animation slides.

**Requirements:** Python 3 with NumPy

```bash
pip install numpy
python train_mlp.py
```

Training takes ~2 minutes on a modern CPU and produces `mnist-sample.js` in the project root. Expected final test accuracy: ~92.6 %.

### 3. Remotion animation dependencies (~624 MB)

The `animations/mlp/` sub-project uses [Remotion](https://www.remotion.dev/) to render the MLP backpropagation video.

```bash
cd animations/mlp
npm install
```

To re-render the video:

```bash
npx remotion render
```

The output `.mp4` files are excluded from the repo due to size. Pre-rendered copies can be shared separately if needed.
